  <h3 align="center">Features</h3>
  <p align="center">
    Online Song Search :mag:<br>
    Streaming Support :musical_note:<br>
    Download Support :arrow_down:<br>
    Play Local / Downloaded Songs Support :open_file_folder: <br>
